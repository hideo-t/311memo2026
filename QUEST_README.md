- **会場**: EVENT SPACE Koriyama #9（郡山市大町1-4-15 郡山ビル地下1階）

## 🎯 ゲーム構成（5ステージ制）

### Stage 0: タイトル画面
- 「第5回 福島の子供たちのために」
- STARTボタンでクエスト開始
- 無料イベント情報を表示

### Stage 1: ミッション受領
**ミッション：#9（会場）に到達せよ**

3つのサブミッション：
1. ① 日付を確定せよ（3/8 日曜）
2. ② 時間を確定せよ（開場11:40 / 開演12:00）
3. ③ "忘れない言葉"を選べ（3.11〜あの日を忘れない〜）

選択肢を押すたびに**情報が記憶される**仕組み。

### Stage 2: ルート選択
地図風UIで会場への道を確認：
- 郡山駅 → 大町 → Koriyama #9
- 3タップで到達
- タップごとに「住所」「会場名」が表示

### Stage 3: 出演者スキャン
出演者カードを図鑑化：
- 12枚のカードをスキャン
- タップでカードが反転
- 3枚スキャンで「コンボ」発動

### Stage 4: 記憶テスト
3問のクイズで記憶を確認：
- Q1: 日付は？（3/8）
- Q2: 開演は？（12:00）
- Q3: 会場名は？（Koriyama #9）

全問正解で「PERFECT」→ ボーナスポイント

### Stage 5: エンディング
- **あなたの合言葉**を生成（例: `FUKUSHIMA-311-0308-#9-M85H60`）
- 特典情報を「宝箱」演出で表示
- QRコード表示（現地確認）
- SNS共有ボタン

## 🎨 デザイン特徴

### ポスターの4色を活かしたUI
- 🟡 **黄**: ミッション（目的）
- 🟢 **緑**: 会場（ルート）
- 🔵 **水**: 時間（日程）
- 🟣 **紫**: 出演者（ステージ）

### レトロゲーム風
- ドット調のフォント
- ネオン風グロー効果
- 8bitサウンドエフェクト

## 🔊 サウンドエフェクト

Web Audio APIで生成：
- **START**: ゲーム開始音
- **CORRECT**: 正解音
- **WRONG**: 不正解音（シェイクアニメーション付き）
- **SCAN**: スキャン音
- **COMBO**: コンボ発動音（3音階）
- **PERFECT**: パーフェクト音（アルペジオ）
- **ENDING**: ファンファーレ
